Sowa — Dylematy Odpowiedzialności

Aplikacja symulująca dylematy moralne. Użytkownik ocenia sytuację na skali -2 do +2 i poznaje konsekwencje swojego wyboru.

Wartości: uczciwość, honor, bezinteresowność, odpowiedzialność.
